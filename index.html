Learning Codex Cloud

This small static page helps verify the full workflow: create a GitHub repository, let Codex update files, and publish the result with GitHub Pages.

Three-step learning checklist

  1. Connect a GitHub repository to Codex Cloud.
  2. Ask Codex to inspect, edit, and summarize the project.
  3. Review the changes, publish the page, and repeat with a real project.
